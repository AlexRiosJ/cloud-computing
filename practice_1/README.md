El proyecto permite el desarrollo de un buscador de imágenes simple que se conecta a tablas de
DynamoDB, sobre las cuales almacena direcciónes de imágenes y etiquetas.

Cada uno de los módulos contiene información precisa sobre lo que se realiza en cada parte del proceso
y en el reporte técnico encontrará mas información sobre cada uno de los módulos.

La aplicación se encuentra desplegada en http://54.196.194.228/

Solo admite la búsqueda de una etiqueta por vez, y en caso de no encontrarse en la base de datos
no se arrojará un mensaje de 0 resultados.
